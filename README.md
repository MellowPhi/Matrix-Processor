# Matrix-Processor

A project to compute matrices of any size.

### Available Functions 
1. Add matrices
2. Multiply matrix by a constant
3. Multiply matrices
4. Transpose matrix
5. Calculate a determinant
6. Inverse matrix
0. Exit
